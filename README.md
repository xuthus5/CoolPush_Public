# CoolPush

CoolPush是一个QQ消息推送服务。其组件如下:

```txt
- MaraiGo		机器人组件
- Server		服务端
- WebClient		客户端
```

## 如何部署

1. 运行MaraiGo机器人组件，其默认提供5700调用端口，自行修改
2. 运行Server服务端，修改`config.yaml`配置文件
3. 启动客户端
